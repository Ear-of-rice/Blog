# Linux命令

## 清空当前面板

```linux
clear
```



## ps-查询运行程序

```
sudo docker ps
```



## images-查看安装程序

```
sudo docker images
```



## cd-去某个文件夹下

```
cd /mydata/
```



## ls-查看该文件下的目录

```
ls
```



## cd ../-返回上一层

```
cd ../
```



## docker ps -a   查询未启动的程序

```
docker ps -a
```



## docker start  ***  启动程序

```
docker start  ***部分即可
```
